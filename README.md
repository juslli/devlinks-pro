# DevLinks Pro

Hub de links profissional com visual moderno em tema dark roxo, feito com HTML, CSS e JavaScript.

## 🚀 Sobre o projeto

O **DevLinks Pro** é uma página de links pessoal inspirada em layouts modernos e elegantes, criada para reunir em um só lugar os principais acessos de um desenvolvedor, como portfólio, GitHub, LinkedIn e projetos.

O foco desse projeto foi praticar:

- estruturação com HTML
- estilização avançada com CSS
- interatividade com JavaScript
- responsividade
- design moderno com tema dark e detalhes em roxo

## ✨ Funcionalidades

- Foto de perfil em destaque
- Nome e descrição profissional
- Botões com ícones
- Efeito de luz acompanhando o mouse
- Layout moderno com glassmorphism
- Responsivo para celular e desktop

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts
